Var :

Var was the original way to declare variables in JavaScript. Veriable declared with var have global scope

let :

let was introduced in ES6. variable declared with let have block scope

const :

const is used to declare constants, whose values cannot be reassigned

null :

null is a special value that represents the intentional absence of any object value. null is often used as a deliberate absence of value

undefined :

undefined is another special value that represents a variable that has not been assigned a value.

REST API :

A REST API is an API that conforms to the design principles of the REST, or representational state transfer architectural style.
